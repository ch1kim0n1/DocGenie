# Output Formats

DocGenie supports markdown and HTML output.

## Markdown (`README.md`)

Generated by `ReadmeGenerator`, includes:

- Project overview and features
- Installation and usage sections
- Project structure summary
- API summary extracted from parsed code

Command:

```bash
docgenie generate . --format markdown
```

## HTML (`docs.html`)

Generated by `HTMLGenerator`, includes:

- Styled documentation layout
- Navigation sidebar with TOC
- Syntax-highlighted code blocks
- README-based or analysis-based rendering

Commands:

```bash
docgenie generate . --format html
# or
docgenie html README.md --source readme --output docs.html
```

## Both

```bash
docgenie generate . --format both
```

## Preview mode

```bash
docgenie generate . --preview
```

Preview prints markdown (or truncated HTML preview) without writing files.
