DocGenieConfig¶

Configuration manager for DocGenie HTML converter.

Supports configuration files in YAML, JSON, and TOML formats. Configuration is loaded in this priority order: 1. Command line arguments (highest priority) 2. Environment variables 3. Project-specific config file (.docgenie.yml, .docgenie.json, .docgenie.toml) 4. User-specific config file (~/.docgenie/config.yml) 5. Default values (lowest priority)
